# Project Instructions
#### Note: You must have python installed to run this project. If "python3" does not work, try using just "python".

## To run main code:
    In your terminal, navigate to the directory where the files are stored. This should be ../Final_Project.
    Once in this directory, run:
        python3 main.py

## To run unit tests:
    In your terminal, navigate to the directory where the files are stored. This should be ../Final_Project.
    Once in this directory, run:
        python3 tests.py
